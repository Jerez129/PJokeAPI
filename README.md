Descripción:  
Esta es una web sencilla de HTML, CSS y JavaScript diseñada para practicar el uso de fetch() 
conectando dos APIs: la PokéAPI, para buscar Pokémon y ver sus stats en una interfaz tipo Pokédex, y la JokeAPI, 
para generar chistes aleatorios. El enfoque principal fue aplicar la estructura que vimos en clase para
manejar datos JSON, manteniendo el código limpio, sin librerías externas y organizado en GitHub.
